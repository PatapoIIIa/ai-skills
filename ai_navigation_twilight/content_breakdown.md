# Content Breakdown

| Axis | Question |
|---|---|
| Delivery | Is it map content, item, spell/action, UI, status effect, job/species/lore, or subsystem behavior? |
| Base | Which existing branch already owns similar behavior? |
| Delta | What changes: stats, visuals, verbs, signals, processing, assets, data? |
| Effect | Who observes it and how often does it run? |
| Risk | Does it touch core, global signals, process loops, or project includes? |

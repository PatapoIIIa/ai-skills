# Content Creation

1. Define audience/frequency.
2. Prefer the narrowest existing host: subtype > local override > component/element > subsystem/global signal.
3. Put Twilight-local additions in `modular_twilight_axis/**` unless a real owner requires core.
4. For desert content, use `modular_deserttown/**` and `_maps/map_files/deserttown/**`.
5. For TGUI content, bind `$ss13-tgui` directly.
